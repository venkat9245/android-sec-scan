# 🔐 Android Security Scan Tool
## 🎯 Project Purpose

Many students and general users install APK files from unknown sources or click suspicious links without understanding the potential security risks.

This project was created to raise awareness and provide a basic security scanning tool that helps users:

- Analyze APK-related risks
- Understand potential security issues
- Perform basic system safety checks
- Encourage safe installation practices

The goal of this tool is educational awareness and preventive security — helping users think before they install.
A lightweight Bash-based security scanning script for Android/Linux environments.  
This tool performs basic security checks and system analysis for educational and authorized testing purposes.

---

## 🚀 Features

- System information gathering
- Security configuration checks
- Permission analysis
- Basic vulnerability indicators
- Lightweight & fast execution

---

## 🛠 Requirements

- Linux / Kali Linux / Android (Termux)
- Bash shell
- Basic system permissions
