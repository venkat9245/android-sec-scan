# 🔐 Android Security Scan Tool

A lightweight Bash-based security scanning script for Android/Linux environments.  
This tool performs basic security checks and system analysis for educational and authorized testing purposes.

---

## 🚀 Features

- System information gathering
- Security configuration checks
- Permission analysis
- Basic vulnerability indicators
- Lightweight & fast execution

---

## 🛠 Requirements

- Linux / Kali Linux / Android (Termux)
- Bash shell
- Basic system permissions

---

Clone the repository:

```bash
git clone https://github.com/venkat9245/android-sec-scan.git
cd android-sec-scan
